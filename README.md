
### Project setup

- Select node's version
  ```bash
  nvm use 14.18.2
  ```

- Clone repo HTTPS
  ```bash
  git clone https://gitlab.com/groupwebrtc/baumarkt/admin.git
  cd admin
  ```

- Install dependencies
  ```
  npm install
  ```

- Values for backend
  ```
  export REACT_APP_API_URL=http://localhost:#####/
  npm start
  ```

### Env and dependencies
